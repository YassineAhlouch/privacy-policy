Demande de Suppression de Données

Chez Odeo Food, nous respectons votre vie privée et votre droit de contrôler vos données personnelles. Cette page explique comment vous pouvez demander la suppression de vos données de nos systèmes.

Procédure de Suppression des Données

Vous pouvez demander la suppression de vos données personnelles à tout moment. Voici comment procéder :

  1. Par email : Envoyez votre demande à privacy@odeofood.com avec pour objet "Demande de Suppression de Données".
  2. Dans l'application : Accédez à votre profil, puis aux paramètres de confidentialité, et cliquez sur "Supprimer mon compte".
  3. Par courrier postal : Envoyez votre demande écrite à notre adresse administrative indiquée ci-dessous.

Veuillez inclure les informations suivantes dans votre demande :

  • Votre nom complet
  • Votre adresse email associée à votre compte
  • Si possible, votre identifiant client
  • Si vous vous êtes connecté via Facebook ou Google, mentionnez-le dans votre demande

Délai de Traitement

Nous traiterons votre demande de suppression de données dans un délai de 30 jours suivant sa réception. Vous recevrez une confirmation une fois que vos données auront été supprimées.

Données Conservées

Veuillez noter que certaines informations peuvent être conservées conformément à nos obligations légales (par exemple, les données relatives aux transactions pour des raisons fiscales). Ces données seront toutefois anonymisées dans la mesure du possible.

Partenaires Tiers

Si vous vous êtes connecté à notre service via des fournisseurs tiers (comme Facebook ou Google), la suppression de vos données dans notre système ne supprime pas automatiquement vos données chez ces fournisseurs. Vous devrez contacter séparément ces services pour demander la suppression de vos données.
